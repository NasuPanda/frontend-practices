# りあクト!

りあクト!シリーズ(技術書)によるReactの学習。

## 目的

- Reactの基礎を理解する
  - 設計思想・哲学
  - 基本的な書き方
  - 現場で使われる書き方
- TypeScriptの基礎を理解する

## 環境

- React(react) 17.0.2
- React DOM(react-dom) 17.0.2
- Create React App(create-react-app) 4.0.3
- TypeScript(typescript) 4.2.3
- ESLint(eslint) 7.14.0
